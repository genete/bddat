Reglas Importantes

NO usar .bg-success para identidad corporativa

El color --bs-success (#198754) es el verde Bootstrap estándar, más claro y "lavado".

Siempre usar .bg-primary para elementos de identidad corporativa (navbar, headers, botones principales).

Usar .bg-primary para identidad

El color --bs-primary (#087021) es el verde corporativo oficial de la Junta de Andalucía.

Es muy cercano al Pantone 356 C oficial (#007a33) y es el correcto para todos los elementos de identidad.

Añadir .bg-gradient para modernizar

Combinar .bg-primary con .bg-gradient añade un efecto de brillo sutil que moderniza la interfaz.

Especialmente recomendado para navbar, headers de cards y botones importantes.

Usar variables CSS en lugar de valores hardcodeados

Siempre que sea posible, usar var(--bs-*) en CSS personalizado para mantener consistencia.

Ejemplo: background-color: var(--bs-primary); en lugar de background-color: #087021;
