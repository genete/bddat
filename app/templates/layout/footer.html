BDDAT
Sistema de tramitación de expedientes · Alta Tensión · v0.3.3

Sistema de tramitación de expedientes de autorización de instalaciones de alta tensión. Desarrollado para la Consejería de Industria, Energía y Minas — Junta de Andalucía.

Servidor

Aplicación web Python con framework Flask y ORM SQLAlchemy. Plantillas Jinja2, migraciones con Alembic.

Base de datos

PostgreSQL como motor relacional. Generación de documentos Word con python-docx y docxtpl.

Interfaz

Bootstrap 5.3 con identidad visual corporativa de la Junta de Andalucía (CDN oficial v1.2.5).

Datos normativos

Repositorio local de legislación consolidada (legalize-es) con acceso a BOE y BOJA.

Integraciones en desarrollo

Diagramas Mermaid.js, componentes React y presentaciones Reveal.js.
